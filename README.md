# LinaHub v3.0

## Upload to GitHub
Upload the **contents** of this folder to the root of the `LinaHub` repository:

- index.html
- manifest.webmanifest
- sw.js
- icon.svg

Do not upload the outer folder as a nested folder.

## Enable GitHub Pages
Repository → Settings → Pages → Deploy from a branch → `main` → `/ (root)` → Save.

Your app address will be:
https://lina3194.github.io/LinaHub/

## Included
- Home dashboard with icon-only navigation
- Plants with watering, notes and photos
- Girls Tank and Boys Tank with the correct residents
- House module tailored to your home and chores
- Medication tracker
- Weight & Health check-ins
- Pokémon GO placeholder
- Dark/light mode
- Backup and restore
- Offline PWA support
