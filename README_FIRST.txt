LINAHUB MODULAR PATCH SYSTEM

These ZIPs are small replacement packages. Each ZIP contains only the files normally needed for that part of LinaHub.

HOW TO USE A PATCH
1. Download the patch ZIP.
2. Extract it on your computer.
3. Open the extracted folder.
4. Upload everything inside it to the ROOT of the LinaHub GitHub repository.
5. Preserve the folder structure and replace files with the same names.
6. Wait for GitHub Pages deployment to finish before uploading another patch.

PATCH MAP
01 Main Home Screen: the large home/menu tile screen.
02 Daily Check-in Journal: journal/check-in feature.
03 Weight and Measures: health weight and measurement tracker.
04 Plants: plant collection, encyclopedia and decorations.
05 Medication: day, medication setup and history.
06 Pokemon GO: friend/gift tracker and filters.
07 Aquariums: aquarium pages and records.
08 House Jobs: rooms, jobs and recurrence.
09 Settings and Theme: settings page, colours and global visual styles.
10 Today Tab: today's task dashboard.
11 To-Do Tab: to-do list and completed tasks.
12 Navigation/Routing/App Shell: default landing pages, bottom navigation, startup, cache and shared data/cloud behaviour.

IMPORTANT
Some visual rules are shared in styles/modules.css. Therefore several feature patches include that same file. A future patch will always be built from the latest agreed baseline so it does not undo earlier changes.

Keep 00_FULL_BASELINE_BACKUP_v15_5.zip somewhere safe. Do not upload it unless a full restoration is needed.
